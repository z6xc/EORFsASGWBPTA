# Full analytic expressions of overlap reduction functions for anisotropies of the stochastic gravitational wave background with pulsar timing arrays

This repository contains the complete analytic expressions for the **anisotropic overlap reduction functions (ORFs) for pulsar timing arrays** derived in our work. All formulas are organized for easy access in a Jupyter environment, allowing researchers and students to quickly browse, copy, or use them.

## Contents

- **Full analytic formulas**: All analytic expressions for anisotropic ORFs, covering various spectral indices, spherical harmonic expansion coefficients, and correlations between pulsar pairs.
- **Jupyter Notebook integration**: All formulas are presented in **LaTeX** within a single notebook (`Rlm_Def_All_SM_eq_smp.ipynb`). After downloading, you can interactively browse the formulas, copy them, or perform numerical calculations.
- **Easy to extend**: The expressions are well‑structured and can be exported as LaTeX code for use in papers or reports.

## Quick start

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```
### 2. Requirements

- Python 3.8 or higher

- Jupyter Notebook or JupyterLab

- Optional: numpy, sympy (for numerical or symbolic computations)

Install dependencies:
```
bash
pip install jupyter numpy sympy
```
### 3. Access the formulas in Jupyter

Start Jupyter:
```
bash
jupyter notebook
```
Click on Rlm_Def_All_SM_eq_smp.ipynb in the file list to browse all formulas with LaTeX rendering and brief explanations.

The notebook is self‑contained; simply open it and you can view, copy, and edit the formulas as needed.

## Formula overview (selected)

This work provides analytic expressions for:

- Spherical harmonic expansion coefficients of the anisotropic ORF

- Kernel functions for different spectral indices

- Explicit relations between pulsar pair geometry and ORF

- Complete orthonormal basis decomposition

For a detailed list of formulas, see the index in the notebook.
